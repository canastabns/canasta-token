# Canasta Token ($CNST)

## Prerequisites

|Dependency|Version|
|:---------|:-----:|
|Node      | 10.24.1 (REQUIRED)|
|NPM       | 6.14.12 |

To run unit tests, clone this repository, and run:

    $ npm install
    $ npm test
